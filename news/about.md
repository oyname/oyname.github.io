I've been fascinated by game development technology since an early age. My focus was often on creating tools to simplify the process, including a self-developed 2D engine that encapsulated DirectX functions to help developers focus on gameplay. With DirectX 7, I delved into 3D programming, which was far more complex. To deepen my understanding, I built my own 3D engine and later uploaded a simple DirectX 11 version to GitHub—a pure hobby project. Curious? Check it out, but keep expectations low. Given today’s demands, a single developer can hardly match modern 3D engines. I'm fascinated not only by game development itself but also by creating 3D models, graphics, and the technology that makes it all possible.

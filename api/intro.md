# Welcome to KROM Engine

KROM is a modern C++20 engine focused on:
- Data-Oriented ECS
- RenderGraph-based rendering
- Modularity through self-registering AddOns

## Who is KROM for?
- Developers looking for a lightweight, extensible engine
- Projects requiring Cross-Platform support (Windows/Linux/macOS)
- Learners who want to understand engine architecture

## What you should know
- C++20 fundamentals
- Basic understanding of ECS & rendering
- CMake build system
